# <a name="f35"></a>F&#35;

_F # 是强类型函数编程语言，专门用于在.NET 上运行_

F # succinctness、 表达和可组合性的类型推断函数编程，强类型化代码的编译时安全和运行时支持、 库、 互操作性、 工具和.NET 对象模型，则将合并。

F # 是现在可用"out 框中的"作为向所有 Xamarin 支持的平台上的 Xamarin 用户编程语言： Android、 OS X 和 iOS。
