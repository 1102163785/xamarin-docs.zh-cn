- **[检查设备方向](device-orientation.md)** &ndash;将指导完成使用本机平台 Api 来确定设备的方向的示例。
- **[获取电池信息](battery-info.md)** &ndash;将指导完成使用本机 Api 以获取有关电池的状态信息的示例。
- **[从库中选取照片](photo-picker.md)** &ndash;将指导完成使用本机 Api 从电话的图片库中选择一张照片的示例。
